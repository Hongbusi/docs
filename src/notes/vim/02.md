# Day 2

## 移动命令

- `0` 移动到行首；
- `^` 移动到本行第一个不是 blank 字符的位置；（改建为 `shift + h`）
- `$` 移动到行尾；
- `g_` 移动到本行最后一个不是 blank 字符的位置。（改建为 `shift + l`）

``` json
"vim.normalModeKeyBindings": [
  {
    "before": ["H"],
    "after": ["^"]
  },
  {
    "before": ["L"],
    "after": ["g", "_"]
  } 
]
```

## 插入命令

- `I` 在光标所在行的行首插入；
- `A` 在光标所在行的行尾插入；
- `O` 在光标所在行的下一行行首插入；
- `o` 在光标所在行的上一行行首插入。

## 复制、粘贴、剪切

- `yy` 复制当前行；
- `p` 粘贴；
- `dd` 剪切当前行。
