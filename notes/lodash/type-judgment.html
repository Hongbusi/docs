# 判断数据类型

# typeof

typeof 操作符返回一个字符串，表示未经计算的操作数的类型。

# 语法

typeof operand
typeof(operand)

# typeof 可能的返回值（8 种）

类型 结果
Undefined "undefined"
Boolean "boolean"
Number "number"
String "string"
Function "function"
Symbol "symbol"
BigInt "bigint"
Object、Null、Array、new Date()、new RegExp() "object"

如想了解更多，可前往 MDN (opens new window) 查看官网文档。
