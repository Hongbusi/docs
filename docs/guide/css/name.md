# CSS 命名规范

## 大模块划分

头部：header

底部：footer

侧边栏：sidebar

悬浮框：popup

单屏：section

弹出层：modal

遮罩层： mask

广告位：banner

## 小模块

logo：logo

导航：nav

电话号码：tel

语言：lang

开关：switch

标题：title

副标题：subtitle

描述：desc

图片：img

背景图片：bg

按钮：btn

列表：list

列表项：list-item

轮播图：swiper

分页：pagination

前缀：prefix

后缀：suffix

占位符：placeholder

图标：icon、symbol

分隔线：divider

表单：form

时间：time、hour、second、minute、year、month、day

评分(星星)：star

点赞：like、unlike

头像：avatar
